
# 📝 파이썬 모듈과 패키지 (PSL & PIP)

코드의 재사용성을 높이고 효율적으로 관리하기 위한 파이썬의 핵심 구조이다.

## 1. 모듈 (Module)

특정 기능을 하는 코드가 작성된 **하나의 파이썬 파일(.py)**이다.
변수, 함수, 클래스 등을 하나의 파일로 묶어놓은 것이다.

### 파이썬 표준 라이브러리 (PSL)

파이썬 설치 시 기본적으로 제공되는 모듈과 패키지의 모음이다. (별도 설치 불필요)

* **`math` 모듈**: 수학 관련 함수 모음 (`pi`, `sqrt` 등)
* **`random` 모듈**: 난수 생성 관련 기능
* **`sys` 모듈**: 시스템 파라미터 및 함수 접근

---

## 2. 모듈 가져오기 (Import) 방법

모듈을 사용하는 방식에 따라 장단점이 다르므로 상황에 맞게 선택해야 한다.

### ① `import 모듈명` (기본)

* **특징**: `모듈명.함수명` 형태로 사용한다. (Dot 연산자 활용)
* **장점**: 이름 충돌을 방지할 수 있어 가장 안전하다.
* **단점**: 코드가 길어질 수 있다.

```python
import math

print(math.pi)      # 모듈명.변수명
print(math.sqrt(4)) # 모듈명.함수명

```

### ② `from 모듈명 import 함수명`

* **특징**: 모듈명을 생략하고 함수 이름만으로 바로 사용한다.
* **장점**: 코드가 간결해진다.
* **단점**: 출처를 알기 어렵고, 기존 변수와 **이름 충돌**이 발생할 수 있다.

```python
from math import pi, sqrt

print(pi)      # math.pi가 아님
print(sqrt(4)) 

```

### ③ `as` 키워드 (별칭 사용)

* **특징**: 모듈이나 함수에 별명을 붙여 사용한다.
* **용도**: 이름이 너무 길거나, 서로 다른 모듈에서 같은 이름의 함수를 가져올 때 충돌을 해결하기 위해 사용한다.

```python
import matplotlib.pyplot as plt  # 긴 이름을 plt로 줄여서 사용
from my_math import sqrt as my_sqrt # 이름 충돌 방지

```

### ⚠️ 주의: 이름 충돌과 와일드카드(*)

* **이름 충돌**: 서로 다른 모듈에서 같은 이름의 함수를 `from`으로 가져오면, **가장 마지막에 import된 것**이 덮어쓴다.
* **와일드카드(`*`)**: `from math import *` 방식은 권장하지 않는다. 어떤 함수가 들어왔는지 알 수 없고 변수 오염이 쉽기 때문이다.

## 3. 패키지 (Package)

연관된 여러 모듈들을 하나의 **디렉토리(폴더)** 에 모아 놓은 것이다.

* **구조**: `패키지(폴더) -> 하위 패키지 -> 모듈(.py)`
* **목적**: 기능별로 파일을 분류하여 관리하기 위함이다.

### 사용자 정의 패키지 구조 예시

```text
my_package/          (패키지)
    statistics/      (하위 패키지)
        tools.py     (모듈)
    math/
        my_math.py   (모듈)

```

```python
from my_package.statistics import tools
print(tools.mod(1, 2))

```

---

## 4. 패키지의 종류와 관리 (PIP)

### 종류

1. **표준 라이브러리 (Built-in)**: `math`, `random`, `json` 등 파이썬 설치 시 포함됨.
2. **외부 패키지 (Third-party)**: 전 세계 개발자들이 만든 패키지. 별도 설치가 필요하다.
* `requests`: 웹 요청/응답 처리
* `pandas`: 데이터 분석 및 엑셀 조작
* `matplotlib`: 데이터 시각화



### 패키지 관리자 (PIP)

외부 패키지를 설치하고 관리하는 명령어 도구이다.

* **설치 명령어**: 터미널에 `$ pip install 패키지명`을 입력한다.

```bash
$ pip install requests

```

### 활용 예시: Requests

웹 브라우저 없이 파이썬 코드로 웹 서버에 요청을 보낼 때 사용한다.

```python
import requests

# 사용 전 반드시 공식 문서를 확인하는 습관을 들여야 한다.
response = requests.get('https://www.google.com')

```
