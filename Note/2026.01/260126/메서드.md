
# 📝 데이터 구조와 메서드 (Data Structure & Method)

데이터를 효율적으로 저장, 관리하고 활용하기 위한 핵심 도구이다.

## 1. 기본 개념

* **데이터 구조**: 데이터를 효율적으로 다루기 위해 구조화한 것 (List, Str, Dict 등). 프로그램의 성능과 유지보수성에 큰 영향을 미친다.
* **메서드 (Method)**: 객체(Object)에 속한 함수이다. `데이터타입.메서드()` 형태로 호출하여 사용한다.
* 예시: `'hello'.upper()` (문자열 객체에게 대문자로 바꾸라고 명령)



---

## 2. 시퀀스 공통 메서드 (Sequence Methods)

문자열과 리스트 같은 시퀀스 데이터에서 공통으로 사용 가능한 탐색 메서드이다.

* **`s.index(x)`**: 항목 `x`가 첫 번째로 나타나는 인덱스를 반환한다. (없으면 **Error** 발생)
* **`s.count(x)`**: 항목 `x`가 등장하는 전체 횟수를 반환한다.

---

## 3. 문자열 메서드 (String Methods)

문자열은 **불변(Immutable)**이므로, 메서드를 사용해도 원본은 변하지 않고 **새로운 문자열을 반환**한다.

### ① 탐색 및 검증 (Inquiry)

* **`s.find(x)`**: `x`의 첫 번째 위치를 반환한다. **(없으면 -1 반환)**
* *비교*: `index()`는 없으면 에러, `find()`는 -1 반환.


* **`s.isalpha()`**: 알파벳으로만 구성되어 있는지 확인한다. (공백, 숫자 포함 시 False)
* **`s.isupper()` / `s.islower()**`: 모두 대문자/소문자인지 확인한다.

### ② 조작 및 변경 (Manipulation) - ★필수 암기

데이터 전처리 과정에서 매우 빈번하게 사용된다.

* **`s.replace(old, new, [count])`**: `old` 문자를 `new` 문자로 바꾼다. `count`를 지정하면 앞에서부터 해당 개수만큼만 바꾼다.
* **`s.strip([chars])`**: 양쪽 끝의 불필요한 공백(기본값)이나 특정 문자를 제거한다.
* **`s.split(sep)`**: 문자열을 `sep` 기준으로 쪼개어 **리스트(List)**로 반환한다. (기본값: 공백)
* **`separator.join(iterable)`**: 리스트 등의 요소들을 `separator`로 연결하여 **문자열(String)**로 반환한다.

```python
# split과 join의 관계
text = "Python is good"
parts = text.split()       # ['Python', 'is', 'good'] (List)
new_text = "-".join(parts) # "Python-is-good" (String)

```

### ③ 대소문자 변환

* **`capitalize()`**: 맨 앞 글자만 대문자 (나머지 소문자)
* **`title()`**: 각 단어의 앞 글자만 대문자
* **`upper()`** / `lower()**`: 전체 대문자 / 전체 소문자
* **`swapcase()`**: 대문자는 소문자로, 소문자는 대문자로 반환

## 4. 리스트 메서드 (List Methods)

리스트는 **가변(Mutable)**이므로, 메서드 사용 시 **원본 자체가 변경**되는 경우가 많다.

### ① 값 추가 (Add)

* **`L.append(x)`**: 리스트 끝에 항목 `x`를 **그대로** 추가한다. (리스트를 넣으면 리스트 자체가 들어감)
* **`L.extend(iterable)`**: 리스트 끝에 다른 반복 가능한 객체의 **모든 요소를 풀어서** 추가한다. (리스트 병합 효과)
* **`L.insert(i, x)`**: 인덱스 `i` 위치에 항목 `x`를 끼워 넣는다.

```python
nums = [1, 2]
nums.append([3, 4]) # [1, 2, [3, 4]] -> 덩어리째 추가
nums.extend([3, 4]) # [1, 2, 3, 4]   -> 요소만 풀어서 추가

```

### ② 값 삭제 (Remove)

* **`L.remove(x)`**: 리스트에서 첫 번째로 나타나는 값 `x`를 삭제한다. (없으면 Error)
* **`L.pop(i)`**: 인덱스 `i`에 있는 값을 **삭제하고 반환**한다. (인덱스 미지정 시 마지막 항목)
* **`L.clear()`**: 리스트의 모든 항목을 삭제하여 빈 리스트 `[]`로 만든다.

### ③ 정렬 및 순서 (Sort & Order)

**주의**: 이 메서드들은 원본을 변경하고, 결과로 `None`을 반환한다.

* **`L.sort()`**: 오름차순으로 정렬한다. (`reverse=True` 옵션 시 내림차순)
* **`L.reverse()`**: 정렬하지 않고 단순히 순서를 뒤집는다.

## 💡 핵심 비교: sort() vs sorted()

많은 입문자가 헷갈려하는 포인트이다.

| 구분 | `L.sort()` | `sorted(L)` |
| --- | --- | --- |
| **소속** | 리스트의 메서드 (`list.sort`) | 파이썬 내장 함수 (`built-in function`) |
| **반환 값** | `None` | **정렬된 새로운 리스트** |
| **원본** | **원본이 변경됨** | 원본은 유지됨 |

```python
data = [3, 1, 2]

# sort() 사용
print(data.sort()) # None 출력 (반환값이 없음)
print(data)        # [1, 2, 3] (원본이 바뀜)

# sorted() 사용
data = [3, 1, 2]
print(sorted(data)) # [1, 2, 3] (새 리스트 반환)
print(data)         # [3, 1, 2] (원본 유지)

```
