# 01 if문 (조건문)

### 1. if문이란 무엇인가?
> 프로그래밍에서 "조건"을 판단하여 상황에 맞는 조치를 취하게끔 만드는 구문입니다.  
> "돈이 있으면 택시를 타고, 없으면 걸어간다"와 같은 상황을 코드로 구현할 때 사용합니다.  

**기본 구조:**
```python
if 조건문:
    수행할_문장1
    수행할_문장2
    ...
else:
    수행할_문장A
    수행할_문장B
    ...
예제 코드:Pythonmoney = True

if money:
    print("택시를 타고 가라")
else:
    print("걸어 가라")
```
---
### 2. if문의 들여쓰기 (Indentation) - ⚠️ 매우 중요
> Python에서 if문에 속한 모든 문장은 반드시 같은 너비로 들여쓰기를 해야 합니다.  
> 들여쓰기가 맞지 않으면 IndentationError가 발생하여 프로그램이 실행되지 않습니다.  
#### 잘못된 예시들:
```Python
# case 1: 들여쓰기 너비가 다름
if money:
    print("택시를")
    print("타고")
        print("가라")  # <--- 에러 발생

# case 2: 일부만 들여쓰기 안 함
if money:
    print("택시를")
print("타고")          # <--- 에러 발생
    print("가라")
```
#### Tip: 공백(Space) 4칸을 사용하는 것을 권장합니다. (탭(Tab) 사용 시 공백과 혼용하지 않도록 주의)

### 3. 조건문이란 무엇인가? (비교 연산자)  
> if 조건문 자리에는 **참(True)**인지 **거짓(False)**인지 판별할 수 있는 문장이 들어가야 합니다.  
> 주로 비교 연산자를 사용합니다.  
#### ⚖️ 비교 연산자 (Comparison Operators)

두 값을 비교하여 참(`True`) 또는 거짓(`False`)을 반환하는 연산자입니다.

| 연산자 | 설명 |
|:---:|:---|
| `x < y` | x가 y보다 작다 (미만) |
| `x > y` | x가 y보다 크다 (초과) |
| `x == y` | x와 y가 같다 |
| `x != y` | x와 y가 같지 않다 (다르다) |
| `x >= y` | x가 y보다 크거나 같다 (이상) |
| `x <= y` | x가 y보다 작거나 같다 (이하) | 
#### 사용 예시:
```Python
x = 3
y = 2

print(x > y)  # True
print(x < y)  # False
print(x == y) # False
print(x != y) # True
```
#### 실전 적용:
```Python
money = 2000

if money >= 3000:
    print("택시를 타고 가라")
else:
    print("걸어가라")
```

### 4. 연산자 - and, or, not
> 조건을 여러 개 연결하거나 판단을 뒤집을 때 사용합니다.  
#### 🧠 논리 연산자 (Logical Operators)

참(`True`)과 거짓(`False`)을 다루는 불리언(Boolean) 연산자입니다.

| 연산자 | 설명 |
|:---:|:---|
| `x or y` | x와 y 둘 중 하나만 참이어도 참이다 |
| `x and y` | x와 y 모두 참이어야 참이다 |
| `not x` | x가 거짓이면 참이다 (반대) | 
#### 사용 예시:
```Python
money = 2000
card = True

# 돈이 3000원 이상 있거나(OR), 카드가 있다면
if money >= 3000 or card:
    print("택시를 타고 가라")
else:
    print("걸어가라")
```
---
### 5. in, not in
> 파이썬에만 있는 독특하고 편리한 연산자입니다.  
> 리스트, 튜플, 문자열 안에 찾으려는 값이 있는지 확인합니다.   
  


#### 🔍 멤버십 연산자 (Membership Operators)

리스트, 튜플, 문자열 등 컬렉션 내부에 특정 값이 존재하는지 확인하는 연산자입니다.

| 연산자 | 설명 |
|:---:|:---|
| `x in 리스트` | 리스트 안에 x가 있는가? (있으면 True) |
| `x not in 리스트` | 리스트 안에 x가 없는가? (없으면 True) |
#### 사용 예시:  
```Python
# 1 in [1, 2, 3]      -> True
# 1 not in [1, 2, 3]  -> False

pocket = ['paper', 'cellphone', 'money']

if 'money' in pocket:
    print("택시를 타고 가라")
else:
    print("걸어가라")
```

### 6. 다중 조건 판단 (elif)
#### 1) elif의 기본 개념과 구조
* elif는 단독으로 쓰일 수 없으며, 반드시 if 문 뒤에 와야 합니다. 기본 구조는 다음과 같습니다.
* if: "만약 ~라면" (가장 먼저 검사)
* elif: "그게 아니고, 만약 ~라면" (if가 거짓일 때 검사)
* else: "위의 모든 조건이 아니라면" (선택 사항, 나머지 모든 경우)

**기본 문법 (Python 예시)**
```Python

score = 85

if score >= 90:
    print("A학점")      # 1. 첫 번째 검사
elif score >= 80:
    print("B학점")      # 2. 1번이 거짓일 때 검사
elif score >= 70:
    print("C학점")      # 3. 2번이 거짓일 때 검사
else:
    print("F학점")      # 4. 위 조건이 모두 거짓일 때 실행
```
#### 2) elif의 핵심 특징 (작동 원리)
**elif를 이해할 때 가장 중요한 것은 순서와 효율성입니다.**
* 순차적 검사 (Top-down): 위에서부터 아래로 차례대로 조건을 확인합니다.
* 조기 종료 (Early Exit): 조건 중 하나라도 참(True)이 되면,  
  그 즉시 해당 코드를 실행하고 전체 조건문을 빠져나갑니다. 그 밑에 있는 elif나 else는 쳐다보지도 않습니다.
* 상호 배타적: if ~ elif ~ else 구조는 여러 선택지 중 단 하나만 실행됨을 보장합니다.

#### 3) 중요 비교: if만 계속 쓰는 것 vs elif를 쓰는 것
이 부분이 초보자들이 가장 많이 헷갈리는 부분입니다. 결과가 완전히 다를 수 있습니다.

**A. elif를 사용한 경우 (택일)**
```Python

n = 10

if n > 5:
    print("5보다 큽니다") # 여기서 참이므로 출력하고 끝!
elif n > 8:
    print("8보다 큽니다") # 실행되지 않음 (이미 위에서 걸러짐)
```
결과: "5보다 큽니다" 하나만 출력됨. (조건들은 경쟁 관계)**

**B. if를 연속으로 사용한 경우 (독립 시행)**
```Python

n = 10

if n > 5:
    print("5보다 큽니다") # 참이므로 출력
if n > 8:
    print("8보다 큽니다") # 별개의 검사이므로 또 확인하고 출력
```
결과: "5보다 큽니다" 와 "8보다 큽니다" 둘 다 출력됨. (조건들이 독립 관계)**

#### 4) 언제 elif를 써야 할까? (실생활 예시)
**"여러 가지 상황 중 딱 하나에만 해당될 때" 사용합니다.**
* 학점 계산: 점수는 하나인데 등급은 A, B, C, D 중 하나여야 할 때.
* 지하철 요금: 나이에 따라 '어린이', '청소년', '성인' 중 하나의 요금만 적용될 때.
* 가위바위보: 내가 낸 것이 '가위', '바위', '보' 중 하나일 때.

**예시: 지하철 요금 계산기**
```Python

age = 15

if age < 8:
    fee = 0          # 미취학 아동
elif age < 14:
    fee = 450        # 어린이
elif age < 20:
    fee = 720        # 청소년 (여기서 걸림 -> fee는 720)
else:
    fee = 1250       # 성인

print(f"요금은 {fee}원 입니다.")
```
**요약**
* if - else : 거짓인 값 출력함.
* if - elif : 거짓인 값 출력하지 않음.
* if - elif(거짓이면 pass) - elif(거짓이면 pass) - else

### 7. 조건부 표현식 (Conditional Expression)if와 else를 한 줄로 간결하게 작성할 수 있습니다.  
#### 구조: 
> 조건문이_참인_경우 if  
> 조건문이_거짓인_경우 else  
```Python
score = 70

# 일반적인 작성법
if score >= 60:
    message = "success"
else:
    message = "failure"

# 조건부 표현식 (한 줄 작성)
message = "success" if score >= 60 else "failure"
```
