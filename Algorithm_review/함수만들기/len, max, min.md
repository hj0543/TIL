


### len
```python
list_a = [3 ,2 ,4 ,6, 7 ,8, 1]
def len1():
    cnt = 0

    for i in list_a:
        cnt += 1
    return cnt
```
### max
```python
list_a = [3 ,2 ,4 ,6, 7 ,8, 1]
def max_():
    max_num = list_a[0]
    for i in range(len1()):
        if list_a[i] > max_num:
            max_num = list_a[i]
    return max_num
```
