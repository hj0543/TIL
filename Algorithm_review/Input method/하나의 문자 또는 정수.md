
# 📝 Python 입력: 단일 데이터 (Single Item) 처리

## 0. 기본 세팅

```python
import sys
input = sys.stdin.readline

```

## 1. 입력값 그대로 받을 때 하나 받을 때

가장 흔한 경우 (예: 문제의 줄 수 `N` 입력)

```python
# 정수 (int는 \n 자동 제거됨)
N = int(input())

# 문자열 (⚠️ 주의: 그냥 받으면 뒤에 \n이 붙음)
S = input().rstrip()
```

## 2. 입력값 나눠서 리스트로 받을 때
1. `input()`만 쓰면 뒤에 줄바꿈(`\n`)이 같이 들어오므로 제거해야 함.    
2. "숫자(int)는 공백을 신경 안 써도 되지만, 문자열(str)은 **오른쪽 공백 제거(rstrip)**를 안 하면 무조건 \n 폭탄을 맞는다!"
### 정수
```python
# 가정: 입력창에 "  123  "이라고 치고 엔터를 누름 (앞뒤 공백 존재)
# 실제 들어오는 값: "  123  \n"

# 1. 굳이 strip()을 쓰고 변환하는 경우 (불필요한 과정)
# num1 = int(input().strip()) 

# 2. 그냥 바로 int()에 넣는 경우 (추천 👍)
# 설명: int()가 알아서 "  ", "\n" 다 무시하고 123만 가져감
num2 = int(input())
print(num2)  # 결과: 123
```

### 문자열
문자열은 \n도 문자로 취급하므로 제거 작업이 필수다.
```python
# 가정: 입력창에 "ABC"라고 치고 엔터를 누름
# 실제 들어오는 값: "ABC\n"

# raw = input()
# print(list(raw))       # 결과: ['A', 'B', 'C', '\n'] (줄바꿈 포함됨)

# 1. rstrip() 사용 (오른쪽만 제거) -> 추천 👍
r_data = input().rstrip()
print(list(r_data))    # 결과: ['A', 'B', 'C'] (깔끔)

# 2. strip() 사용 (양쪽 다 제거)
s_data = input().strip()
print(list(s_data))    # 결과: ['A', 'B', 'C'] (결과는 같음)

```

## 3. (심화) 진짜 딱 '한 글자'만 읽을 때 -> 어지간해서 그냥 readline 써도 됨

`readline`은 한 줄을 통째로 읽지만, `read(1)`은 바이트 단위로 딱 1개만 읽어옴. 입력 데이터가 거대하고 문자 하나하나 처리해야 할 때 가장 빠름.

```python
# 입력이 'A'일 때
char = sys.stdin.read(1) 
# 결과: 'A' (줄바꿈 문자는 읽지 않고 버퍼에 남음)

```

## 4. 리스트로 감싸서 받을 때 (List) -> 스택, 큐, DP초기값 정도?

단일 데이터지만, 이후 로직 편의를 위해 리스트 형태 `[값]`이 필요할 때.

```python
# 숫자 하나를 리스트로: [5]
N_list = [int(input())]

# 문자열 하나를 리스트로: ['hello']
S_list = [input().rstrip()]

```
