
### 1. 필수 및 제어 (가장 중요)

개발 속도를 높여주는 핵심 단축키입니다.

| 기능 | Windows / Linux | macOS | 설명 |
| --- | --- | --- | --- |
| **전체 검색** | `Shift` 두 번 | `Shift` 두 번 | 파일, 클래스, 설정 등 모든 요소 검색 |
| **명령어 검색** | `Ctrl` + `Shift` + `A` | `Cmd` + `Shift` + `A` | 기능 이름으로 검색하여 실행 (메뉴 위치 몰라도 됨) |
| **퀵 픽스 (Quick Fix)** | `Alt` + `Enter` | `Opt` + `Enter` | 모듈 임포트, 에러 수정, 코드 개선 제안 |
| **최근 파일** | `Ctrl` + `E` | `Cmd` + `E` | 최근 작업한 파일 목록 팝업 |

### 2. 코드 편집 (Editing)

파이썬 코드 작성 시 유용한 편집 키입니다.

| 기능 | Windows / Linux | macOS | 설명 |
| --- | --- | --- | --- |
| **줄 복제** | `Ctrl` + `D` | `Cmd` + `D` | 현재 줄을 아래로 복사 |
| **줄 삭제** | `Ctrl` + `Y` | `Cmd` + `Delete` | 현재 줄 삭제 |
| **코드 포맷팅** | `Ctrl` + `Alt` + `L` | `Cmd` + `Opt` + `L` | PEP8 등 스타일에 맞춰 코드 자동 정렬 |
| **들여쓰기/내어쓰기** | `Tab` / `Shift` + `Tab` | `Tab` / `Shift` + `Tab` | 블록 선택 후 들여쓰기 수준 조절 |
| **파라미터 정보** | `Ctrl` + `P` | `Cmd` + `P` | 함수 괄호 안에서 인자(Parameter) 정보 보기 |
| **주석 처리** | `Ctrl` + `/` | `Cmd` + `/` | `#` 주석 토글 |

### 3. 탐색 및 이동 (Navigation)

방대한 프로젝트에서 원하는 위치로 빠르게 이동합니다.

| 기능 | Windows / Linux | macOS | 설명 |
| --- | --- | --- | --- |
| **정의로 이동** | `Ctrl` + `B` | `Cmd` + `B` | 변수나 함수가 정의된 곳으로 이동 |
| **사용처 찾기** | `Alt` + `F7` | `Opt` + `F7` | 해당 변수/함수가 사용된 모든 곳 검색 |
| **파일 찾기** | `Ctrl` + `Shift` + `N` | `Cmd` + `Shift` + `O` | 파일 이름으로 검색하여 이동 |
| **뒤로/앞으로 가기** | `Ctrl` + `Alt` + `←/→` | `Cmd` + `[/]` | 이전 커서 위치로 이동 (매우 유용) |

### 4. 리팩토링 (Refactoring)

변수명 변경 등을 안전하게 수행합니다.

| 기능 | Windows / Linux | macOS | 설명 |
| --- | --- | --- | --- |
| **이름 변경 (Rename)** | `Shift` + `F6` | `Shift` + `F6` | 변수/함수 이름을 참조된 모든 곳에서 일괄 변경 |
| **변수 추출** | `Ctrl` + `Alt` + `V` | `Cmd` + `Opt` + `V` | 선택한 값을 변수로 생성 |
| **함수(메소드) 추출** | `Ctrl` + `Alt` + `M` | `Cmd` + `Opt` + `M` | 선택한 코드 블록을 함수로 분리 |

### 5. 실행 및 디버깅 (Run & Debug)

| 기능 | Windows / Linux | macOS | 설명 |
| --- | --- | --- | --- |
| **현재 파일 실행** | `Ctrl` + `Shift` + `F10` | `Ctrl` + `Shift` + `R` | 현재 열려있는 파이썬 파일 실행 |
| **이전 실행 반복** | `Shift` + `F10` | `Ctrl` + `R` | 마지막으로 실행했던 설정 다시 실행 |
| **디버그 실행** | `Shift` + `F9` | `Ctrl` + `D` | 디버그 모드로 실행 |
| **브레이크포인트** | `Ctrl` + `F8` | `Cmd` + `F8` | 디버깅 중단점(Breakpoint) 설정/해제 |

### 💡 팁: Keymap 설정 확인

만약 위 단축키가 작동하지 않는다면, `File` > `Settings` (Mac은 `PyCharm` > `Settings`) > **Keymap** 설정이 **Windows**나 **macOS** 기본값으로 되어 있는지 확인해보세요. (가끔 Eclipse나 VS Code 키맵으로 설정된 경우가 있습니다.)
