
### 1. 필수 및 제어 (가장 중요)

이 단축키들은 인텔리제이의 모든 기능을 제어하는 핵심입니다.

| 기능 | Windows / Linux | macOS | 설명 |
| --- | --- | --- | --- |
| **전체 검색 (Search Everywhere)** | `Shift` 두 번 | `Shift` 두 번 | 클래스, 파일, 설정, 액션 등 모든 것을 검색 |
| **액션 검색 (Find Action)** | `Ctrl` + `Shift` + `A` | `Cmd` + `Shift` + `A` | 메뉴 위치를 몰라도 기능 이름으로 실행 가능 |
| **퀵 픽스 / 제안 (Show Intentions)** | `Alt` + `Enter` | `Opt` + `Enter` | 에러 수정, 코드 개선 제안 확인 (매우 중요) |
| **최근 파일 열기** | `Ctrl` + `E` | `Cmd` + `E` | 최근에 열었던 파일 목록 팝업 |

### 2. 검색 및 이동 (Navigation)

코드를 빠르게 탐색할 때 사용합니다.

| 기능 | Windows / Linux | macOS | 설명 |
| --- | --- | --- | --- |
| **클래스 찾기** | `Ctrl` + `N` | `Cmd` + `O` | 클래스 이름으로 검색하여 이동 |
| **파일 찾기** | `Ctrl` + `Shift` + `N` | `Cmd` + `Shift` + `O` | 파일명으로 검색하여 이동 |
| **선언부로 이동** | `Ctrl` + `B` | `Cmd` + `B` | 변수나 메소드가 정의된 곳으로 이동 |
| **구현부로 이동** | `Ctrl` + `Alt` + `B` | `Cmd` + `Opt` + `B` | 인터페이스의 실제 구현체로 이동 |
| **뒤로/앞으로 가기** | `Ctrl` + `Alt` + `←/→` | `Cmd` + `[/]` | 커서 위치 히스토리 따라 이동 |
| **파일 구조 보기** | `Ctrl` + `F12` | `Cmd` + `F12` | 현재 파일의 메소드/변수 목록 팝업 |

### 3. 코드 편집 (Editing)

코딩 속도를 높여주는 편집 기능입니다.

| 기능 | Windows / Linux | macOS | 설명 |
| --- | --- | --- | --- |
| **줄 복제** | `Ctrl` + `D` | `Cmd` + `D` | 현재 줄을 아래로 복사 |
| **줄 삭제** | `Ctrl` + `Y` | `Cmd` + `Delete` | 현재 줄 삭제 |
| **구문 완성** | `Ctrl` + `Shift` + `Enter` | `Cmd` + `Shift` + `Enter` | 괄호, 세미콜론 등을 자동으로 닫고 줄바꿈 |
| **코드 포맷팅** | `Ctrl` + `Alt` + `L` | `Cmd` + `Opt` + `L` | 코드를 설정된 스타일대로 자동 정렬 |
| **줄 단위 이동** | `Shift` + `Alt` + `↑/↓` | `Opt` + `Shift` + `↑/↓` | 현재 줄을 위/아래로 이동 |
| **주석 처리** | `Ctrl` + `/` | `Cmd` + `/` | 한 줄 주석 (`//`) 토글 |

### 4. 리팩토링 (Refactoring)

변수명 변경이나 코드 구조 변경 시 안전하게 사용합니다.

| 기능 | Windows / Linux | macOS | 설명 |
| --- | --- | --- | --- |
| **이름 변경 (Rename)** | `Shift` + `F6` | `Shift` + `F6` | 변수, 파일, 메소드 이름을 참조된 곳까지 일괄 변경 |
| **변수 추출** | `Ctrl` + `Alt` + `V` | `Cmd` + `Opt` + `V` | 선택한 값을 변수로 추출 |
| **메소드 추출** | `Ctrl` + `Alt` + `M` | `Cmd` + `Opt` + `M` | 선택한 코드 블록을 메소드로 분리 |

### 5. 실행 및 디버깅 (Run & Debug)

| 기능 | Windows / Linux | macOS | 설명 |
| --- | --- | --- | --- |
| **실행 (Run)** | `Shift` + `F10` | `Ctrl` + `R` | 이전에 실행했던 설정으로 앱 실행 |
| **디버그 (Debug)** | `Shift` + `F9` | `Ctrl` + `D` | 디버그 모드로 실행 |
| **스텝 오버 (Step Over)** | `F8` | `F8` | 디버깅 시 다음 줄로 이동 |
| **스텝 인투 (Step Into)** | `F7` | `F7` | 디버깅 시 메소드 내부로 진입 |

### 💡 팁: 단축키를 빨리 익히는 방법

1. **Key Promoter X 플러그인 설치:** 마우스로 클릭할 때마다 해당 기능의 단축키를 화면에 띄워주는 플러그인입니다.
2. **단축키 검색:** 기능을 쓰고 싶은데 단축키를 모를 땐 `Shift` + `Shift`를 누르고 기능 이름을 검색하면 옆에 단축키가 표시됩니다.
